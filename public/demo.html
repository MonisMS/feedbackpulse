FeedbackPulse Widget Demo

Look at the bottom-right corner of your screen!

You should see a blue "Feedback" button. Click it to test the widget.

About This Demo

This page demonstrates the FeedbackPulse feedback widget.

Features:

  • ✅ Floating feedback button
  • ✅ Modal popup with form
  • ✅ Feedback type selection (Bug/Feature/Other)
  • ✅ Optional email and name fields
  • ✅ Form validation
  • ✅ Success/error messages
  • ✅ No external dependencies
  • ✅ Responsive design

How to use:

  1. Create a project in your FeedbackPulse dashboard
  2. Copy the embed code
  3. Paste it into your website before the closing </body> tag
  4. The feedback button will appear automatically!

Note: Replace YOUR_PROJECT_KEY in the script below with your actual project key from the dashboard.
